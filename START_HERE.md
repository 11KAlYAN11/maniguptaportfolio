## 🚀 Deploy in 3 Easy Steps

### Step 1: Push to GitHub (if not already done)
```bash
cd /home/pavan/javaPS/maniguptaportfolio
git add .
git commit -m "Portfolio website - ready to deploy"
git push origin main
```

### Step 2: Go to Vercel and Deploy (Takes 2-5 minutes)
1. Visit [vercel.com](https://vercel.com)
2. Click "Sign Up" → Sign in with GitHub
3. Click "New Project"
4. Select `maniguptaportfolio` repository
5. Click "Deploy"

✅ **Your site is now live!** Check Vercel dashboard for your URL

### Step 3: Optional - Add Custom Domain
1. Buy domain (Namecheap, GoDaddy, Google Domains, etc.)
2. Vercel Dashboard → Project → Settings → Domains
3. Add your domain and update DNS records
4. Wait 24 hours for DNS propagation

**That's it! Your portfolio is live on the web! 🎉**
